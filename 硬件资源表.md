# 硬件资源表

## 芯片与开发环境
- **芯片型号**：MSPM0G3507，LQFP-64
- **内核**：Arm Cortex-M0+
- **主频**：32 MHz SYSOSC，MCLK 不分频
- **开发框架**：TI MSPM0 DriverLib + SysConfig
- **IDE**：Keil MDK / CCS SysConfig
- **NVIC 分组**：Cortex-M0+ 固定优先级模型
- **最后更新**：2026-07-10

| DMA_CH0 | UART0 RX | 外设到内存 | UART RX | 由 BSP UART 配置 |

## 中断优先级矩阵

| 中断源 | 优先级 | 理由 |
|--------|--------|------|
| I2C0_IRQn | 0 | MPU6050 异步事务时序 |
| GROUP1_IRQn / GPIOB | 0 | 两路编码器边沿计数，优先于调试 UART 以减少漏采 |
| UART0_IRQn | 1 | 调试通信与 DMA 状态处理 |
| SysTick_IRQn | 由 BSP 时基配置 | 系统毫秒时基 |

## 时钟配置

| 时钟源 | 频率 | 用途 |
|--------|------|------|
| SYSOSC / MCLK | 32 MHz | CPU 与总线时钟 |
| TIMG8 | 32 MHz / 1 | 电机 PWM |
| I2C0 | BUSCLK | MPU6050，400 kHz 总线 |
| UART0 | BUSCLK | 调试串口，115200 bps |
