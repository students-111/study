# 全国大学生电子设计竞赛赛题索引

> 把全国大学生电子设计竞赛历年赛题映射到本 skill 的具体场景与示例，让"我要做 2023D"能秒级定位到入口。
>
> **范围**：本 skill 只覆盖控制 / 计算 / 底层驱动类题目。视觉相关赛题（含摄像头、赛道识别、目标追踪）由独立 `auto-vision` skill 承担。

---

## 0. 怎么使用本索引

1. **按"我在做什么"查表**：从下面 6 个表挑一个匹配你赛题类型的表
2. **找到对应的"推荐入口"列**：跳到那个 mode 或 ref 文件读
3. **不确定题目分类的**：看 §7 决策树

如果你在做的题目本表没列，参考 §8"未列入题目的处理流程"。

## 7. 决策树：你的题目不在表里？

```
我的题目主要做……
│
├── 输出一个信号 → §1 信号源类
│
├── 测一个信号的指标 → §2 仪表类
│
├── 已调信号的解调 / 测量 → §3 调制类
│
├── 有噪声要"学着"去除 → §4 信号处理（自适应）
│
├── 控制系统（PID / 平衡 / 跟踪）→ §5 控制类
│   也可以参考 主线场景 4-5 + lqr-example-*
│
├── 模拟电路 / 电源 → §6 模电类
│
├── 有摄像头 / 图像识别 → 调用独立 `auto-vision` skill（不在本索引）
│
└── 我不知道 → 详细描述题目，让 embedded-dev 路由
```

## 8. 跨场景组合题（电赛常见）

电赛历年题型多数是**两到三个能力组合**，下面给"组合解法"参考：

| 题型 | 组合 | 示意 |
|---|---|---|
| DDS + 失真度自检 | E1 + E3 | 设计自带 THD 测量的信号源 |
| 调制 + 测量 | E1 + E2 + E3 | 完整通信发射机 + 自检指标 |
| 电磁 + 控制 | 主线 4-5 | 电磁循迹小车（无视觉） |
| 信号分析 + 自适应 | E2 + E5 | 含未知干扰的解调 |

## 9. 推荐学习路径（按年级）

### 大一：基础熟悉
1. `.auto-embedded/refs/matlab-hello-5min.md`（5 分钟跑通）
2. `.auto-embedded/refs/matlab-example-step-id.md`（认识传函）
3. `.auto-embedded/refs/matlab-example-iir-filter.md`（滤波器）

### 大二：单一场景深入
1. E1 DDS（自己做一个信号源）
2. E3 仪表（做一个频谱仪）
3. 主线场景 4 控制器（做一个 PID/LQR）

### 大三/大四：竞赛实战
1. 按题选场景组合
2. 一键流水线（`.auto-embedded/modes/matlab-firmware-pipeline.md`）
3. MIL/SIL/PIL 验证链（主线场景 10）

## 10. 与主协议的衔接

竞赛场景下推荐的 RIPER-5 走法：

```
RESEARCH：本索引找题型 + 看历年开源代码（gh search）
INNOVATE：在 E1-E7 + 主线 1-10 间挑组合
PLAN：列实施清单（含 MATLAB 脚本 + .h 导出 + 嵌入式集成 + 测试方案）
EXECUTE：用流水线 mode 跑完整链路
REVIEW：对照赛题评分点 + arch-check + 静态分析
```
